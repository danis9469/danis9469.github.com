# 廚師料理冊交接檔

## 專案目的

這個資料夾是一個可直接打開的單頁 HTML 美食料理網頁。主入口是 `index.html`，頁面名稱為「廚師料理冊｜Chef's Recipe Book」。

使用者可以依序選擇：

1. 國家或地區：臺灣、中國、美國、義大利、日本、韓國、泰國、越南、德國、俄國。
2. 用餐時段：早餐、午餐、晚餐。
3. 對應料理：每個國家至少 15 道，頁面會顯示簡單介紹、所需食材、製作流程、料理對應授權照片與影片教程連結。

## 檔案說明

- `index.html`：主要網頁，包含 HTML、CSS、JavaScript 與料理資料，不需要額外打包。
- `server.cjs`：可選的 Node.js 靜態伺服器，用於本機預覽。
- `HANDOFF.md`：本交接檔，供另一台電腦上的 Codex 或開發者快速接手。
- `danis.html`：舊檔案，與目前美食網頁無關，可以保留作備份。

## 如何在另一台電腦執行

最簡單方式：

1. 將整個資料夾複製到另一台電腦。
2. 直接用瀏覽器打開 `index.html`。

如果瀏覽器對本機檔案有安全限制，可在此資料夾啟動簡單伺服器：

```powershell
python -m http.server 8787 --bind 127.0.0.1
```

然後打開：

```text
http://127.0.0.1:8787/index.html
```

如果沒有 Python，也可以用 Node.js 建立靜態伺服器，或直接雙擊 `index.html`。

本資料夾已附 Node.js 伺服器，可執行：

```powershell
node server.cjs
```

預設網址是：

```text
http://127.0.0.1:8793/index.html
```

## 功能檢查清單

接手後請確認：

1. 首頁標題顯示「廚師料理冊」。
2. 點擊「臺灣、中國、美國、義大利、日本、韓國、泰國、越南、德國、俄國」會切換國家料理。
3. 點擊「早餐、午餐、晚餐」會切換對應食譜。
4. 料理下拉選單與快速料理卡片可切換同時段不同料理。
5. 食材清單、製作流程與料理授權照片會隨選項更新。
6. 原本的影片嵌入區已移除，避免嵌入影片黑畫面；每個步驟旁都有 YouTube 影片教程搜尋連結。
7. 步驟區不再放不準確的圖片框；沒有明確對應圖就只顯示文字步驟與影片教程。
8. 下方 Chef's Recipe Book 會自動輪播料理，點擊卡片可切換到該料理。
9. 手機寬度下內容不會重疊，料理卡片與流程區會改為上下排列。

## 修改料理資料的位置

所有料理資料都在 `index.html` 底部的 JavaScript 物件 `menus` 裡。

每道料理包含：

- 第 1 欄：料理名稱。
- 第 2 欄：短描述。
- 第 3 欄：料理風格或菜系。
- 第 4 欄：食材清單。
- 第 5 欄：製作流程。

`countries` 管理國家按鈕與簡介；`licensedPhotos` 管理已確認的 Wikimedia Commons 授權圖片白名單。只有白名單內的料理會顯示圖片，避免使用不相關的隨機圖片。

## 注意事項

- 料理圖片使用 Wikimedia Commons 授權圖片的 `Special:FilePath` 連結；需要網路才能載入照片。
- 沒有確認授權與內容相符的料理，不會顯示隨機圖片。
- 目前沒有直接嵌入 YouTube iframe，避免黑畫面；影片教程以外開 YouTube 搜尋連結呈現。
- 本頁沒有使用 npm、React、Vite 或後端服務，維護時優先保持單檔 HTML 架構；`server.cjs` 只負責本機預覽。

# 🍜 Interactive Global Cuisine Experience

## 📖 專案簡介

本專案旨在展示如何運用 HTML、CSS 與 JavaScript，
打造一套具備「互動式體驗」的異國料理食譜系統。

使用者可透過國旗與國家選單，
探索不同國家的代表料理，
並查看：

- 料理介紹
- 熱量資訊
- 食材重量
- 運動消耗時間
- 臺灣版料理做法
- 臺灣 YouTube 教學影片

本系統強調：

- 視覺化互動體驗
- 動態內容切換
- 美食卡片動畫
- 國家分類瀏覽
- 沉浸式料理探索介面

---

# 🎯 專案核心 Skill

本專案主要展示以下前端開發技能：

## 1️⃣ HTML 結構設計

透過 HTML 建立：

- Hero Banner
- 國家選擇介面
- 料理卡片系統
- 動態內容區塊
- 食譜資訊區域

展示完整網站結構規劃能力。

---

## 2️⃣ CSS 動畫與視覺設計

本專案大量使用：

- Hover 動畫
- 卡片浮動效果
- 漸層按鈕
- 陰影層次
- 響應式排版
- Cinematic UI 設計

打造具備「影片感」的美食瀏覽體驗。

---

## 3️⃣ JavaScript 互動邏輯

透過 JavaScript 完成：

- 國家切換功能
- 動態內容顯示
- Smooth Scroll 滾動
- 使用者互動切換
- 卡片動態載入

提升網站互動性與操作體驗。

---

## 4️⃣ 異國料理資料整合能力

系統整合：

- 日本料理
- 韓國料理
- 義大利料理
- 台灣料理

並加入：

- 熱量估算
- 公克數計算
- 運動消耗時間
- 臺灣料理做法

讓網站更接近真實美食平台。

---

# 🌏 功能特色

## ✅ 國家互動選單

使用者可透過：

- 國旗圖示
- 中文國家名稱

快速切換不同國家料理。

---

## ✅ 動態料理卡片

每道料理皆包含：

- 專屬料理圖片
- 熱量資訊
- 公克數
- 運動消耗
- 食譜做法
- YouTube 教學連結

---

## ✅ 臺灣料理教學整合

所有影片皆改為：

「臺灣影片料理做法」

避免出現國外語言教學，
更符合台灣使用者需求。

---

## ✅ 響應式設計（Responsive Design）

支援：

- 電腦版
- 平板
- 手機版

自動調整版面配置。
